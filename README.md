## Modules

You can find what modules are included in Python:
[Python Module Index](https://docs.python.org/3/py-modindex.html)

Ways to import them:

```python
import math
# use it
math.pow()

from math import pow
# use it
pow()

# set an alias
import numpy as n
```

How to install modules that aren't comming by default

```shell
uv add numpy
```
